# Movie Sorter
A simple Java application designed to manage a movie library and sort items by rating
# Project Structure
- `Media.java` (Parent Class): Defines general media attributes
- `Movie.java` (Child Class): Extends the `Media` class to demonstrate inheritance
- `SmartLibrary.java` (Main Class): Acts as the program`s entry point
